# Laboratório 3 – Alinhamento, Homografia 2D e Mosaico

**Visão Computacional 2026.2**

## Descrição

Este repositório contém o relatório e os códigos do Laboratório 3 da disciplina de Visão Computacional, focado em **alinhamento de imagens**, **homografia 2D** e **Image Stitching (mosaico)** utilizando OpenCV.

## Conteúdo

- `Lab3_Homografia.ipynb` — Relatório completo em formato Jupyter Notebook
- `stitching_webcam.py` — Script standalone para homografia e mosaico em tempo real com duas webcams
- `imagens/` — Imagens utilizadas nos experimentos e resultados gerados

## Requisitos

```bash
pip install opencv-python opencv-contrib-python numpy matplotlib jupyter
```

## Execução

### Notebook
```bash
jupyter notebook Lab3_Homografia.ipynb
```

### Script Webcam (requer duas webcams)
```bash
python stitching_webcam.py
```
